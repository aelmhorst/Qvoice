Release Notes: Qvoice Version 3.0

Release Date ?

Here is a summary of the most recent changes that have been made to Qvoice

The name of the person creating the PO ( perhaps their extension as well ) will be listed on the PO
The Order Open screen will be color-coded:
Black = New Order
Red = Materials on a PO
Green = Materials received
You will be able to go directly to the PO screen for an order, even if it is in non-edit mode
A  new report will list all Open POs and backordered items
A new login screen will come up when Qvoice is started ( the previous username and password will be saved ). This will allow for the following:
We can remove editing of the “Salesperson” field on the order screen
There will be logins for each office worker and a special “shop” login
If the “shop” login is used, the user will be presented with a simplified screen that shows only tasks that can be done from the shop.
The only initial task available to the shop login would be the ability to “Check In Purchase Orders”
A new PO Check in screen will have the following features
A list all open POs
Selecting a PO and marking it as “Received”
Ability to deselect individual items and mark them as backordered
This screen will be “touch-screen friendly”, which means it will be meant to be used without a keyboard.
