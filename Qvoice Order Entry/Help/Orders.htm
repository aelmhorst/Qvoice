Creating an order

Related topics:

Entering Square Edge Special Tops

To create an order:

  1. Click the FILE -> NEW menu (Or CTRL-N).
  2. The Order Entry Screen will show.
  3. Select a customer from the list on the left side of the screen.
  4. Fill in any other relevant information.
  5. Click the "Line Details" tab at the top of the Order Entry Screen (Or ALT-L).
  6. Enter a length for the first slab.
  7. Select a slab type from the list.
  8. Enter any charges in the Additional Charges field. See Entering Charge Codes for details.
  9. Enter part of the color code or name and press enter.
  10. The system will look up any color matches.
  11. Select the appropriate color from the list.
  12. Click the "Add" button to add the line (Or TAB and then Enter from the color field).
  13. Repeat as necessary until all countertops are entered.
  14. Save the order by clicking FILE -> Save (Or CTRL-S).
  15. Close the Order Entry screen using the standard Window close box at the upper right hand of the screen (Or CTRL-F4).

Hints:

Qvoice will automatically prompt you to save the order if you attempt to close an order with unsaved changes on it.
Using the Tab key to move between fields and the keyboard shortcuts noted above will result in faster, more efficient entry of orders.
If you wish to delete a detail line, click on it and press the "Delete" key.
If you wish to edit a detail line, click on it three times in rapid succession. The screen will change to reflect an edit state. Note that the charges will need to be re-entered. Make your changes and then click the "Change" button.

Entering Charge Codes

To enter charge codes:

  1. Enter each code separated by the charges separator. The charges separator defaults to the forward slash (/), but may be changed on a computer as per the user's normal preference.
  2. If you cannot remember a code for a particular charge you can run the Charge Code Lookup Report:
    Click Tools -> Reports -> Charge Code Lookup (Or press F2).
    The report viewer will load with the charge code report. For more information on the report viewer, click here.
    Browse the list for your desired code.
    Or, enter part of the charge code or description in the small text box at the top of the report. Click the binoculars or press Enter and the report will move to any matches of the word in succession.
    Note that charges are grouped into two categories, which are navigable using the group viewer on the left hand side of the report. Normal Charges are the standard charges which are priced using the quantities entered. Lineal Charges will automatically use the lineal or square footage of the countertop to calculate a per-foot charge price.

Entering Square Edged Special tops

To enter Square Edged Special tops:

  1. In the Length field of the line item, enter both dimensions of the countertop separated by an asterisk (*). (e.g. for a 48 inch by 24 inch countertop, enter 48*24 in the Length field. This is also the syntax used for sheets of laminate.
  2. If the countertop has any other special attributes, such as backer, laminate on bottom, or is thicker than standard thickness, those attributes must be entered as charges. Look in  the Lineal Charges section of the charge codes report to find these charges. Finished edges and other charges are entered using normal charge syntax as well.

Opening an order

To open an Order:

  1. Click File -> Open (Or CTRL-O).
  2. In the resulting screen, click on the appropriate order type from the buttons on the left.
  3. In the buttons along the bottom, select the appropriate letter that matches the first letter of the customer that the order is for.
  4. The screen will show you any orders that meet the criteria.
  5. You may sort this screen by clicking on the top of the column you wish to sort by.
  6. Double click the desired order or click to select and then click the "Open Button".
  7. The Order Entry Screen will load with the appropriate order.

Hints:

If an order has partially shipped, you will not be able to edit the order, but you will be able to print an updated delivery receipt.
If you open an order which has been partially shipped, as you look at the detail lines, the backordered lines will show the number of items that have been shipped. Hint: to see any orders that are currently open with backorders on them, click CTRL-O to open the order selection screen. Scroll over to the right in the order listings and sort by the "vchStatusDescription" field. Orders with backorders on them will show up as "In Process"

Printing a delivery receipt

  1. Open an order or enter a new one.
  2. While the order is still open and at the forefront on screen, click FILE -> Print Delivery Receipt (Or CTRL-P).
  3. The delivery receipt will automatically print.

Printing a quote / order confirmation

  1. Open an order or enter a new one.
  2. While the order is still open and at the forefront on screen, click FILE -> Print quote / order confirmation (Or F6).
  3. The report will then print.
  4. Alternatively, you can preview the quote / order confirmation by clicking FILE -> Preview quote / order confirmation (Or F5).
  5. The preview screen has an printer icon at the top that will allow you to print it after previewing, if you so desire. For more information on the report viewer, click here.

Posting an order

Posting an order is the process of letting the system know how much of an order was delivered, whether it was a pickup or delivery, and the date of the delivery. Posted orders later get put on invoices that are mailed or e-mailed to the customer.

Related topics:

Posting a shipment

To Post an order:

  1. Click TOOLS -> Posting -> Order (Or F11).
  2. In the resulting screen, click on the appropriate order type from the buttons on the left.
  3. In the buttons along the bottom, select the appropriate letter that matches the first letter of the customer that the order is for.
  4. The screen will show you any orders that meet the criteria.
  5. You may sort this screen by clicking on the top of the column you wish to sort by.
  6. Double click the desired order or click to select and then click the "Open Button".
  7. The order posting screen will then load.
  8. This screen contains a list of each line item on the order as shown below.
  9. It shows the amount of the item that was ordered, the amount that was shipped previously, and the amount that you have delivered now is in the "to ship" field.
  10. If any of the items on any of the detail lines was backordered, click on the line and then type the amount delivered in the "amount shipped now" field.
  11. Click the "Change button. The "to ship" field will change to reflect the amount you entered.
  12. If the item was picked up by the customer instead of being delivered to the customer, click the "Pick Up" field on the lower left hand corner.
  13. Enter the delivery date and the name of the person who signed for the order. If that person has signed for an order previously, their name will show up in the list. If there is a delivery charge, enter it here as well.
  14. After all of the information is complete, click the "Post Order" button. The system will respond by asking you if the information is correct. If it is, click "Yes" (Or Enter).
  15. The system will then officially post the order and give you a message of the results.
  16. Click "OK" and you will be returned to the main Qvoice Screen.
  17. If there were any backorders on the order, you may wish to print the new delivery receipt at this time.

Deleting an order

To Delete an order:

  1. Click TOOLS -> Delete  -> Delete Order.
  2. In the resulting screen, click on the appropriate order type from the buttons on the left.
  3. In the buttons along the bottom, select the appropriate letter that matches the first letter of the customer that the order is for.
  4. The screen will show you any orders that meet the criteria.
  5. You may sort this screen by clicking on the top of the column you wish to sort by.
  6. Double click the desired order or click to select and then click the "Open Button".
  7. The system will then respond with a verification message asking if it is OK to delete the specified order.
  8. Clicking "Yes" (Or Enter) will cause the order to be deleted.
