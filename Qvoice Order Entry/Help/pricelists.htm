Pricing Documentation

Database Tables associated with pricing:

This document can be used as a starting point for understanding how the database tables in Qvoice are used to determine prices. These are the tables associated with the complete set of pricing rules inside Qvoice.

 

SlabType

The slab type table groups slabs together that are priced using the same minimum length restrictions. Each price list can place a slab into different length restrictions. For example, STCs have one set of minimum length rules for wholesale pricing and another set of rules for Wausau Homes pricing.. These types differ only by their length restrictions. Different types of slabs have different allowable and mimimum length rules. Note that slab types are specific to price lists. In other words, an STC could be a slab type of  “Normal Customer Slab” in one price List and WHSlab in a different price list.

Column Name

Description
vchSlabTypeDescription A Description of this type of slab, useful for maintaining the database
iSlabTypeID The slab type for this slab and price list.
vchAvailableLengths A comma-seperated lists of allowable lengths
flMinimumLength The minimum length allowed for this type

The mapping of slabs to slab types is done in the SlabPrice table, which is documented below.

SlabPrice

The SlabPrice table holds a dual role in defining pricing for slabs. First of all, it contains per-foot pricing for all of the slabs. Secondly, it links a slab to a specific slab type, which is specific to a price list. This is how different customers can have different rules for minimum footages.

Column Name

Description
IPriceListID A link to the price list for this row
ISlabID A link to the slab in the Slab table
flPrice The per-foot price for this slab and price list
ISlabTypeID The slab type for this slab and price list.

 

ChargePrice

The ChargePrice table determines the price for each charge, based on the current price list.

Column Name

Description

IPriceListID

A link to the price list for this row

IchargeID

A link to the Charge table for this row

FlPrice

The amount charge for this charge

 

LaminatePrice

The laminate price table determines the upcharge for laminates for each price list. Pricing is based upon both laminate type and slab type. Therefore, this table creates a row for every combination of laminate type (stock, red, etc) and slab type (STC, BT, etc).

Column Name

Description

iPriceListID

A link to the price list for this row

iPriceCodeID

A link to the LaminateUpcharge table

islabType

The type of slab that is being priced

mSlabUpCharge

The upcharge for this type of slab and this laminate type

 

LaminateBrand

The laminate brand table defines pricing per customer by brand. If pricing rules mean that specific price lists have top upcharges per laminate brand, that mapping can be done here. This table was necessary for Wausau homes top and job upcharges, but may not have any different rules from the wholesale pricelist in the future.

Column Name

Description

iPriceListID

A link to the price list for this row

vchBrandCode

A code identifying this brand

vchBrandDesc

A description of this brand

flJobUpcharge

The upcharge for an entire job of this color

flTopUpcharge

The upcharge for a single top in this color
