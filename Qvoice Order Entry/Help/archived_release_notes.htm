Release Notes: Qvoice Version 2.5.625

Release Date 01/12/2003

Here is a summary of the most recent changes that have been made to Qvoice

 
Order Entry Screen

The Order Entry screen has a new list at the top of the screen that shows the list of Purchase Orders that have been created for this order:

To go to the editing screen for a PO, select the PO from the list and click the eyeglass button. Also, there will be an item in the list called “Create New . . .”. This will allow the program to create a new PO for this order if there are items on the order that have not been placed on a PO.

 
Purchase Order Editing Screen

You can double-click on a line in the “Details View to go directly to the order.

 
Purchase Order Creation

This screen allows you to create new Pos and to edit existing ones. 

 

You may create a blank PO. The PO will have the requested date that is entered above in the “Purchase Order Date field”. Clicking on “Create a blank PO” will bring up a new blank PO in the PO editor.

 

To create a new slab order, you can enter a date and then click the “Analyze” button to see the results. This will bring up the Slab Order Creation Dialog as show below:

 

 

This screen allows you to see the orders that are ready to be placed on a slab order up to the date you entered on the first screen. At the bottom of the screen, you can see the total footage that is ready to placed on the order.

You can choose “Remove Order” to remove the order from consideration for this slab order. This will remove the order from this list, but will still allow the order to come up the next time you generate a slab order.

You can also choose “Go To Order” to bring up the order editing screen for that order for more details.

 

When you are satisfied with the order, you can choose “Create Slab Order” to create the slab order and bring it up in the PO Editing screen.

 

Click Cancel to close the screen.

 

Looking up Purchase Orders

Back to the “Create or Search for PO” screen . . .

 

 

This screen allows you to print out any purchase orders that were created within the specified dates. It also allows you to edit a purchase order if you enter the PO Number. (Note: you can enter “221” to edit purchase order “PORD000221”

 

Editing Pos

While editing PO’s, you have a new option to add a line item to a PO and then manually tie the PO Line item to an order line item. To do this, first select the row you want to map to an order. The Tools -> Map Po Line Item to Order menu item at the top of the screen will start the process. It will first bring up a job search dialog, showing eligible orders. Once you select an order, the program will ask you which Order Line item to tie the PO Line item to. If the program finds a line number that matches and doesn’t have any other PO item tied to it, it will tie the current PO line item to the order item you selected.

Miscellaneous

The Options screen has a new option that allows you to disable on the splash screen. On a fast computer system, the splash screen is not really necessary as the program loads very quickly.

Release Notes: Qvoice version 2.2.1071

Release Date: 08/20/2001
LabelPrinting

Labels can be printed by pressing CTRL-L at any time or by going to File -> Print -> Labels
Labels can be printed for an order or for a shipment

Angle Calculator

The new angle calculator will allow you to accurately calculate the three angles of a triangle, given its three sides.
This is useful for measuring angles on templates, existing countertops, or for jobsite measurements.
This method of calculating angles is much more accurate than using an anglefinder.

Release Notes: Qvoice version 2.1.1056

Date: 04/19/2001

The "Enter" key can now be used on the "Job Details" screen to move between fields. If you keep hitting "Enter" it will also automatically take you to the "Line Details" screen.

The request date field will now be highlighted when you tab into it so that you can begin typing immediately.

Tip:

When entering a job, if you are in the Requested Date or Entry date fields, you can use the "+" and "-" keys on the numeric keypad to advance or subtract the date by a day. It's much faster than typing in the date. You can also use the "*" and "/" keys on the numeric keypad to advance or subtract the date by a week.

tips to remember (they will make your job easier):

In the Order - > Open dialog box hold down the "CTRL" key and press the "+" key on the numeric keypad one or two times until all of columns collapse to their narrowest readable width. This helps A LOT when you are trying to lookup an order. Unfortunately, there is no way to get that screen to default to that view programmatically, I've tried.

When you are in the customer name field on the order entry screen or on the Invoice lookup screen, it is much faster to just type the first few letters of the customer's name rather than clicking on it with the mouse. Also, for any other fields that use the drop down menu that you see there, such as the OrderType and Salesperson on the JobDetails screen or the SlabColor field on the Line Details screen, the up and down arrows will allow you to make a selection, which may be faster than using the mouse.

pricing is now also doing the right thing. When you enter a 121.25" stc with an endsplash will show up as a 10' slab. When you exit, save, and re-open the order, it will still come up correctly as a 10' slab and prices it as such on the invoice.

Invoice searching has been improved. You can now search invoices by the following:

Invoice Number or Order Number

View all Invoices for a specific date

Or, look up invoices by Customer and filter by PO or Job Number. Enter the first part of the PO or Job number in this field, and only invoices that have Orders with PO's or Job Numbers that begin with what you typed will be displayed.

shipped orders now cannot be deleted from the system and the corresponding Invoice will not be lost. If you attempt to delete an order which has been partially shipped, the system will automatically delete the undelivered portion of the order and do the proper thing to preserve the invoice data associated with the shipped part.

The Invoice Generation Summary report and the Invoice Summary report both now show Orders under each invoice. They also show PO Number, Job Name, and number of items for each order.
