Creating a shipment

A shipment is a group of orders that are delivered together on the same day. A shipment may contain orders from multiple customers. All of the delivery receipts for the shipment can be printed together, grouped and sorted by customer; and all of the orders can be posted in one group, saving large amounts of time.

The system creates and groups shipments based on the following criteria:

Only orders that have the "Put on Shipment" flag checked on the Order Entry Screen. This flag can be automatically preset based on the customer, if the customer is set up with that preference.
All orders for a shipment have to have the same Requested Date.

To create a shipment:

  1. Click TOOLS - > Shipment -> Generate Shipment (Or CTRL-G).
  2. A screen will come up that will ask you the desired delivery date. Orders that have a Request Date that matches this date and the "Put on Shipment" flag will be placed on the shipment.
  3. You may elect to pick all customers or only a single customer for this shipment.
  4. Click "Generate Shipment" (ALT-G).
  5. The system will ask you "Generate Shipment?"
  6. If you click "Yes", the shipment will be generated.
  7. The system will give you a message with the shipment number.
  8. The system will ask you if you want to print the delivery receipts now for the shipment. You may print the delivery receipt later if you so choose.
  9. If you click "Yes", the delivery receipts will be printed.
  10. The Shipment Generation screen will remain open in case you have other shipment to generate.
  11. When you are finished generating shipments, you may exit the shipment generation screen by clicking "Close" (ALT-C) or by using the standard Windows close window button.

Reprinting a shipment

To reprint a shipment:

  1. Click TOOLS -> Shipments -> Reprint Shipment
  2. Enter the Batch ID. The batch ID can be seen on the Order Entry screen of any order that is on the shipment (lower right hand corner) or on a copy of the shipment delivery receipt.
  3. The delivery receipts will be printed to a report viewer so that you can see the orders on the shipment. For more information on the report viewer, click here.
  4. If you wish to print the shipment, click the printer icon at the top of the preview window.

Posting a shipment

Related topics:

Posting an order

To post a shipment:

  1. If there are any backorders anywhere on the shipment, the orders on the shipment that have backorders on them must be posted separately before the shipment can be posted. See posting an order for details.
  2. After any backorders have been posted separately,  the shipment may be posted.
  3. Click TOOLS -> Posting -> Post Shipment (Or F12).
  4. Enter the shipment number and the date the shipment was delivered. The shipment number can be found on the delivery receipt.
  5. Click the "Post Shipment" button.
  6. If there are many orders on the shipment, the posting process could take several minutes.
  7. When all of the orders on the shipment have been automatically posted, a message will appear.
  8. Clicking the OK button on the message will return you from the shipment posting process.

Deleting a shipment

Once a shipment has been created, orders cannot be added to it or removed. A shipment can be deleted, however, and then regenerated, which will create the desired affect of adding or removing orders from it. Deleting a shipment does no harm to the orders on it.

To delete a shipment:

  1. Click TOOLS -> Shipment -> Delete Shipment.
  2. Enter the shipment number.
  3. If you do not know the shipment number, it is printed on the delivery receipt and is also listed on the order entry screen for any order on the shipment (in the lower right hand corner).
  4. You will be asked if you are sure you want to delete the shipment.
  5. Clicking "OK" will delete the shipment.
